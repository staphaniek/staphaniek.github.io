# bundle(jekyll) 설치

```
bundle install
```

# 로컬에서 실행하기

홈 디렉토리에서
```
bundle exec jekyll serve
```
커멘드를 실행하고,
```
localhost:4000
```
에 들어가면 로컬에서 작성 내용을 확인할 수 있다.
